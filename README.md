Supermarket Sales - Exploratory Data Analysis (EDA)

This project is part of the CodeAlpha Data Analytics Internship. It involves analyzing a supermarket sales dataset to uncover insights using Python and Power BI.

Tools Used
-Python (Pandas, Matplotlib, Seaborn)
- Google Colab
- Power BI

Objectives
- Perform Exploratory Data Analysis (EDA)
- Visualize key metrics such as product line distribution, gender-wise sales, time-based patterns, etc.
- Build an interactive Power BI dashboard

Files Included
- EDA.ipynb – Python-based EDA notebook (Google Colab)
- EDA-Dashboard.pbix – Power BI interactive dashboard

Key Insights
- Sales were evenly distributed across product lines
- Slightly higher female customer engagement
- Time-based sales patterns (rush hours, weekdays vs weekends) revealed
